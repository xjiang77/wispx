## Project: agent-wisp

Research and documentation repository. No runnable code — only markdown docs, architecture notes, and learning materials.

## Writing Style

- 中英文混合：技术术语用英文，解释说明用中文
- 文档结构清晰，善用 headers 和 bullet points
- 代码示例用 fenced code blocks，标注语言

## Git Workflow

- Branch: work directly on `main` (solo research repo)
- Commit messages: 中文描述，prefix with `feat/fix/docs/refactor`
- Don't create PRs for this repo — direct push
