# Skill Toolkit

A comprehensive Claude Code plugin for the full agent skill lifecycle â€” from creation to discovery, review, audit, and packaging.

## Skills Included

| Skill | Description |
|-------|-------------|
| **skill-creator** | Create new skills from scratch and iteratively improve them with evals and benchmarks |
| **find-skills** | Discover and install skills from the open agent skills ecosystem |
| **skill-reviewer** | Review individual skills for quality, best practices, and effectiveness |
| **skill-auditor** | Audit skill collections for security, consistency, cross-platform compatibility |
| **skill-packager** | Package skills into Claude Code plugins with versioning, commands, and GitHub sharing |

## Commands

| Command | Description |
|---------|-------------|
| `/create-skill` | Scaffold and build a new skill interactively |
| `/find-skill` | Search for and install skills from the ecosystem |
| `/review-skill` | Run a quality review on an existing skill |
| `/audit-skills` | Audit a collection of skills against standards |
| `/package-plugin` | Package one or more skills into a distributable plugin |

## Installation

```bash
git clone <repo-url> skill-toolkit
claude plugin add ./skill-toolkit
```
